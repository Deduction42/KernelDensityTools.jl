#=======================================================================================================
Leave-One-Out Maximum Likelihood Algorithm 
=======================================================================================================#
1.  H = initial bandwidth
2.  Build interpolated KDE with initial bandwidth kde_H = kde(H, x) (callable object)
3.  Evaluate k0 = k(H, 0)/N
4.  Calculate log-likelihood (predicted kde with ith element removed)
    -   loglik = sum(xi->log(kde_H(xi)-k0), x)
    -   No need to scale by N/(N-1) due to constant offset from log 
5.  Iterate over a bounded univariate search algorithm
